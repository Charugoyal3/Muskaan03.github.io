HTML refrence

Tags List

<!DOCTYPE html>

- this tag defines the document type. Example: <!DOCTYPE html> <html> <head> <title>Title of the document </title> </head>

<!--...-->

-this tag defines a comment. Example: <!--This is a comment. Comments are not displayed in the browser-->

<p>

-this tag defines a paragraph. Example: <p >This is some text in a paragraph.</p>

<video>

-this tag defines a video or movie. Example: <video width="320" height="240" controls> <source src="movie.mp4" type="video/mp4"> <source src="movie.ogg" type="video/ogg"> Your browser does not support the video tag. </video>

<tittle>

-this tag defines a title for the document. Example: <title>HTML Reference</title>

<style>

-this tag defines style information for a document. Example: <html> <head> <style> h1 {color:red;} p {color:blue;} </style> </head> <body>

<b>

-this tag defines bold text. Example: <p> this is a normal text -<b> this is a bold text</b></p>

<header>

-this tag defines header for a document or section. Example: <header> <h1>Most important heading here</h1> <h3>Less important heading here</h3> <p>Some additional information here</p> </header>

<h1> to <h6>

-this tag defines HTML headings. Example: <h1>This is heading 1</h1> <h2>This is heading 2</h2> <h3>This is heading 3</h3> <h4>This is heading 4</h4> <h5>This is heading 5 </h5> <h6>This is heading 6</h6>

<img>

-this tag defines a image. Example: <img src="smiley.gif" alt="Smiley face" height="42" width="42">

<div>

-this tag defines a section of a document. Example: <div style="background-color:lightblue"> <h3>This is a heading</h3> <p>This is a paragraph.</p> </div>

<body>

-this tag defines the body of a document. Example: <html> <head> <title>Title of the document</title> </head> <body> The content of the document...... </body> </html>

<footer>

-this tag defines the footer of the document or section. Example: <footer> <p>Posted by: Hege Refsnes</p> <p>Contact information: <a href="mailto:someone@example.com"> someone@example.com</a>.</p> </footer>

<a>

-this tag define a hyperlink. Example: <a href="https://www.w3schools.com">Visit W3Schools.com!</a>

<li>

-this tag defines a list item. Example: <li>Coffee</li> <li>Tea</li> <li>Milk</li>
